## 🖥프로젝트 실행법🖥
1. 가상환경 생성 및 실행
- `python -m venv (가상환경 이름)`
- `(가상환경 이름)/Scripts/activate`

2. Django 설치
- `pip install django`

3. 서버 실행
- `python manage.py runserver`
